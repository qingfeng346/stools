## 功能介绍
* 命令行压缩解压文件
* szip.exe -op [zip压缩(默认) unzip解压] -zip [类型 (lz4 zip gzip) (必须)] -input [文件路径 (必须)] -output [导出文件路径 (必须)]
### 支持压缩格式
- lz4
- zip
- gzip
